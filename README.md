Simple RESTful API for managing a library system.
Build with FasAPI and PostgreSQL

Requirements:
- Python 3.11+
- PostgreSQL 16+
- Dependencies (see `requirements.txt`):
  - fastapi
  - psycopg[binary]
  - uvicorn

Setup

git clone https://github.com/herner-werzog/recruitment_task
cd recruitment_task
docker compose up
